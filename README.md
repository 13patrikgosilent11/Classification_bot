Этот бот создан для того, чтобы вы могли опредилить вид гриба, если вы точно не уверены, что за гриб вы нашли.
Чтобы данный бот заработал, вы должны загрузить свою фотку, на котором отчетливо виден гриб, и бот через несколько секунд даст вам ответ.
